# Day 02 - CREATE DATABASE

## Objective

Learn how to create databases in PostgreSQL.

## Theory

A database is a container that stores tables and other database objects.

A PostgreSQL server can contain multiple databases.

## Syntax

```sql
CREATE DATABASE database_name;
```

## Example

```sql
CREATE DATABASE school_db;
```

## Key Takeaways

- A PostgreSQL server can have multiple databases.
- CREATE DATABASE creates a new database.
- Use meaningful database names.
